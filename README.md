# Easy HTML Reader

[← Read the German version](README-de.md)  

**Simple HTML reader designed for iPads – best used as a PWA**  

## Description

**Easy HTML Reader** is a minimalist, modern HTML file viewer optimized for **iPads** and touch devices. It allows you to open HTML files or ZIP archives directly in the browser and view their content without installing extra software.  

Especially useful: the app works as a **Progressive Web App (PWA)**, so it can be installed on your iPad like a native app.  

---

## Features

- **HTML and ZIP support** – load single HTML files or ZIP archives with multiple HTML files.  
- **File preview** – view the HTML code in an overlay before opening it.  
- **Live Code Editor** – type your own HTML and see it rendered instantly.  
- **Dark/Light Mode** – automatically adapts to system preferences.  
- **Smooth animations & intuitive UI** – gentle transitions, hover effects, scrollable overlays.  
- **Optimized for touch & iPad** – large buttons, swipe-friendly interface.  
- **Minimalistic & clean** – no unnecessary menus, everything works in-browser.  

---

## Installation & Usage

1. **Open on iPad:**  
   Open `index.html` directly in Safari or another browser.  

2. **Install as PWA (recommended):**  
   - Tap the **Share icon** in Safari.  
   - Select **“Add to Home Screen”**.  
   - The app now launches like a native application without browser UI.  

3. **Load files:**  
   - Drag HTML files or ZIP archives onto the upload area, or tap to select files.  
   - Tap a list item to open the file in the viewer.  
   - Use the **Preview button** to see the HTML code before opening.  

4. **Live Code:**  
   - Tap the **Live Code button** at the bottom right.  
   - Enter HTML and click **Run** to see the result immediately in the viewer.
