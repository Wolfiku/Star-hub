# Easy HTML Reader
[← Read the English version](README.md)  
## Beschreibung

**Easy HTML Reader** ist ein minimalistischer, moderner HTML-Dateibetrachter, der speziell für **iPads** und Touch-Geräte optimiert ist. Er erlaubt es dir, HTML-Dateien oder ZIP-Archive direkt im Browser zu öffnen und den Inhalt zu betrachten, ohne zusätzliche Software installieren zu müssen.  

Besonders praktisch: die Anwendung funktioniert **als Progressive Web App (PWA)**, sodass sie wie eine native App auf deinem iPad installiert werden kann.  

---

## Funktionen

- **HTML- und ZIP-Unterstützung** – lade einzelne HTML-Dateien oder ZIP-Archive mit mehreren HTML-Dateien.  
- **Dateivorschau** – zeige den Inhalt einer HTML-Datei direkt in einem Overlay an, bevor du sie öffnest.  
- **Live Code Editor** – gib eigenen HTML-Code ein und teste ihn direkt im Browser.  
- **Dark/Light Mode** – automatisch passend zum Systemmodus.  
- **Animationsfreundlich & intuitiv** – sanfte Übergänge, Hover-Effekte und Scroll-freundliche Overlays.  
- **Optimiert für Touch & iPad** – große Buttons, Swipe-freundliche UI.  
- **Minimalistisch & übersichtlich** – keine unnötigen Menüs, alles direkt im Browser.  

---

## Installation & Nutzung

1. **Auf iPad öffnen:**  
   Öffne die `index.html` direkt im Safari oder einem anderen Browser.  

2. **Als PWA installieren (empfohlen):**  
   - Tippe auf das **Teilen-Symbol** in Safari.  
   - Wähle **„Zum Home-Bildschirm“**.  
   - Die App startet jetzt wie eine native Anwendung ohne Browser-UI.  

3. **Dateien laden:**  
   - Ziehe HTML-Dateien oder ZIP-Archive auf das Upload-Feld oder tippe darauf, um Dateien auszuwählen.  
   - Tippe auf einen Listeneintrag, um die Datei im Viewer zu öffnen.  
   - Nutze den **Preview-Button**, um den HTML-Code vorher zu sehen.  

4. **Live Code:**  
   - Tippe auf den **Live-Code-Button** unten rechts.  
   - Gib HTML ein und tippe auf **Run**, um das Ergebnis sofort im Viewer anzuzeigen.

## Vorteile für iPad-Nutzer

- Keine Installation von komplizierten Entwicklungswerkzeugen notwendig.  
- Offline-fähig als PWA – ideal für unterwegs oder in der Schule.  
- Touch-optimierte Bedienung: große Buttons, Scrollbare Vorschau.  
- Perfekt für schnelle Tests oder zum Lernen von HTML auf iPad.

## Technische Details

- **HTML, CSS & JavaScript** – keine Backend-Abhängigkeiten.  
- **JSZip** – für die Verarbeitung von ZIP-Dateien im Browser.  
- **Progressive Web App ready** – kann offline und als eigenständige App genutzt werden.  
- **Dark/Light Mode** – automatisch, basierend auf Systemeinstellungen.
